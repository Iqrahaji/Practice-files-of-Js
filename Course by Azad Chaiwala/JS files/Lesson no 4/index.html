Array and Objects

Data type Description Example
Array An array is a special variable, which can hold more than one value at a time. let fruits =
["Apple", "Banana", "Mango"];
// index number: [0] , [1] , [2]
console.log(fruits[1]); // Output: Banana
Object An object is a collection of properties, and a property is an association between a name (or key) and a value. let person = {
firstName: "John",
lastName: "Doe",
age: 30
};
console.log(person.firstName); // Output: John
