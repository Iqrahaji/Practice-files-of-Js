Array and Objects

Array

An array is a special variable, which can hold more than one value at a time.

1=> if we want to change the value of an array

2=> if we want to add a new value to an array

3=> if we want to remove the last value from an array

4=> if we want to remove the first value from an array

5=> if we want to add a new value to the beginning of an array

6=> if we want to find the length of an array

let fruits =

["Apple", "Banana", "Mango"];

=> index number: [0] , [1] , [2]

console.log(fruits[1]); => Output: Banana

1=> fruits[1] = "Orange";

console.log(fruits); => Output: ["Apple", "Orange", "Mango"]

2=> fruits.push("Pineapple");

console.log(fruits); => Output: ["Apple", "Orange", "Mango", "Pineapple"]

3=> fruits.pop();

console.log(fruits); => Output: ["Apple", "Orange", "Mango"]

4=> fruits.shift();

console.log(fruits); => Output: ["Orange", "Mango"]

5=> fruits.unshift("Strawberry");

console.log(fruits); => Output: ["Strawberry", "Orange", "Mango"]

6=> console.log(fruits.length); => Output: 4

Object

An object is a collection of properties, and a property is an association between a name (or key) and a value.

1=> changing the age property

2=> adding a new property to the object

3=> removing a property from the object

let person = {

firstName: "John",

lastName: "Doe",

age: 40

};

console.log(person.firstName); => Output: John

1=> person.age = 41;

console.log(person.age); // Output: 41

2=> person.city = "New York";

console.log(person.city); // Output: New York

3=> delete person.lastName;

console.log(person); // Output: {firstName: "John", age: 41, city: "New York"}
