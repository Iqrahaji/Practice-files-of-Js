How to Link Js to your Html file.

There are three ways.

Method Description Example
Inline js Writing JavaScript code directly within HTML elements using event attributes.

<button onclick="alert('Hello World!')">Click Me</button>

By script tag Embedding JavaScript code within <script> tags in the HTML file.

<script>alert('Hello World!')</script>

Link with file Linking an external JavaScript file to the HTML document using the <script> tag with the src attribute.

<script src="script.js"></script>
