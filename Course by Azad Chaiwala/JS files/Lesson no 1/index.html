How to Link Js to your Html file.

There are three ways.

  • Inline js
  • Link with file.
  • By script tag.

Inline js

The js that write in html.

Example:

<button onclick="alert('Hello World!')">Click Me</button>

By script tag

The js that write inside script tag in html file.

Example:

<script>alert('Hello World!')</script>

Link with file

The js that link with html file.

Example:

<script src="script.js"></script>
