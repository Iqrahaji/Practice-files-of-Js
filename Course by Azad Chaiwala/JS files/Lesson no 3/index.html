Data Types

There are five data types.

Type Description Example
String variables Any thing that write in the quotes(""). let name = "Iqra haji"
console.log(typeof name)
Boolean variables In which only two conditions occurs. True or False. let pakistanWin = "true"
console.log(typeof pakistanWin)
Null variables It's value is null. It's typeof always console as object. let phoneNumber = null
console.log(typeof phoneNumber)
Number variables Variables value assigns to a number. let age = 25
console.log(typeof age)
undefined variables Variables value is not assigned. let address;
console.log(typeof address)
