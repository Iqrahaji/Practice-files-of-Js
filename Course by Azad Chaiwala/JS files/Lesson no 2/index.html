Variables and Data types

variables(var, let, const)

Variable Description Example:
var Global Scope or Function Scope

var name = "Azad Chaiwala"; console.log(name); // Azad Chaiwala

let Block Scope

let age = 22; console.log(age); // 22

const Block Scope & cannot be reassigned

const pi = 3.14; console.log(pi); // 3.14
